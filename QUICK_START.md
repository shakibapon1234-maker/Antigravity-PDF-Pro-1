# 🚀 QUICK START - Antigravity PDF Pro

## ⚡ সবচেয়ে দ্রুত উপায়

### Step 1: চালান (Recommended)
```
dist/win-unpacked/ ফোল্ডারে যান
"Antigravity PDF Pro.exe" ডাবল ক্লিক করুন
```

### Step 2: অথবা সরাসরি
```
dist/win-unpacked/Antigravity PDF Pro.exe চালান
```

---

## 📂 Desktop এ কীভাবে রাখবেন (সবচেয়ে সাধারণ সমস্যা)

### সঠিক উপায়:
```
1. এই ফোল্ডার খুঁজুন:
   D:\Main Branch\Antigravity-PDF-Pro-1\dist\win-unpacked

2. সম্পূর্ণ "win-unpacked" ফোল্ডার কপি করুন
   (শুধু .exe নয়!)

3. Desktop এ পেস্ট করুন

4. Desktop এর পেস্ট করা ফোল্ডারে ডাবল-ক্লিক করুন

5. "Antigravity PDF Pro.exe" ডাবল-ক্লিক করে চালান
```

### যা করবেন না:
```
❌ শুধু .exe ফাইল কপি করবেন না
❌ শুধুমাত্র কিছু .dll ফাইল কপি করবেন না
❌ ফাইলগুলি আলাদা জায়গায় রাখবেন না
```

## 🛠️ যদি এরর আসে?

### Error: "ffmpeg.dll not found"
→ পুরো ফোল্ডার একসাথে কপি করেছেন কিনা চেক করুন

### Error: "procedure entry point"
→ ডাউনলোড করুন: https://aka.ms/vs/17/release/vc_redist.x64.exe

### এখনও কাজ করছে না?
```
চালান: SETUP.bat
বেছে নিন: Option 3 (Fix: Run from Desktop)
নির্দেশনা অনুসরণ করুন
```

---

## 📦 অন্যদের কাছে পাঠাতে চান?

```
1. চালান: CREATE_DISTRIBUTION.bat
2. এটি তৈরি করবে: "Antigravity-PDF-Pro-Portable" ফোল্ডার
3. এই ফোল্ডার ZIP করুন
4. অন্যদের পাঠান
5. তারা Extract করে RUN.bat চালাবে
```
