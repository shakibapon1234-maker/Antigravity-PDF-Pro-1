# ANTIGRAVITY PDF PRO — WEBSITE & LICENSING PLAN

এই পরিকল্পনাটি মূলত **Antigravity PDF Pro** ডেস্কটপ অ্যাপ্লিকেশনের সেলস ল্যান্ডিং পেজ এবং লাইসেন্স ভ্যালিডেশন সার্ভার (Licensing Server) সেটআপ করার জন্য তৈরি করা হয়েছে।

---

## ১. হোস্টিং, ডাটাবেস ও ডোমেন সেটআপ

### ক) ডোমেন নেম (Domain Name)
* **সফটওয়্যার ডোমেন:** `antigravitypdf.com` (বা `antigravitypdfpro.com`)। এটি একটি গ্লোবাল সফটওয়্যার ডাউনলোড ও প্রোডাক্ট সেলস পেজ হিসেবে কাজ করবে।

### খ) হোস্টিং ও ডাটাবেস আর্কিটেকচার
1. **ল্যান্ডিং পেজ (Frontend):** 
   - **Vercel** / **Netlify** (সম্পূর্ণ ফ্রি হোস্টিং)। এটি অ্যাপের ফিচার ও টিউটোরিয়াল ল্যান্ডিং পেজ হোস্ট করবে।
2. **লাইসেন্স সার্ভার (Backend API):**
   - **Render** বা **Railway** প্ল্যাটফর্মে একটি Node.js/Express সার্ভিস হোস্ট করা হবে (ফ্রি টায়ার থেকে শুরু করে পরবর্তীতে $5/মাস)।
   - কাস্টমারদের অ্যাক্টিভেশন ডাটা এবং লাইসেন্স কী সংরক্ষণের জন্য একটি হালকা **SQLite** বা **Supabase PostgreSQL** ডাটাবেস ব্যাকএন্ডের সাথে যুক্ত থাকবে।

## ২. লাইসেন্সিং ও পেমেন্ট অটোমেশন (Stripe & Local Gateways)

ব্যবহারকারী যখন অনলাইনে সফটওয়্যারটি কিনবেন, পেমেন্ট সফল হওয়ার সাথে সাথে স্বয়ংক্রিয়ভাবে তার ইমেইলে লাইসেন্স কি চলে যাবে।

### ক) পেমেন্ট গেটওয়ে ইন্টিগ্রেশন:
* **Stripe (আন্তর্জাতিক কাস্টমারদের জন্য):** বিশ্বব্যাপী ক্রেডিট বা ডেবিট কার্ডের মাধ্যমে পেমেন্ট নেওয়ার জন্য।
* **SSLCommerz / Shurjopay (স্থানীয় কাস্টমারদের জন্য):** বিকাশ, রকেট বা নগদ দিয়ে পেমেন্ট নেওয়ার জন্য।

### খ) পেমেন্ট ও লাইসেন্সিং ওয়ার্কফ্লো (Webhook Automation):
```
১. কাস্টমার ওয়েবসাইটে পেমেন্ট করবেন
       ↓
২. পেমেন্ট গেটওয়ে থেকে সার্ভারে "Webhook" রিকোয়েস্ট আসবে
       ↓
৩. লাইসেন্স সার্ভার একটি ইউনিক লাইসেন্স কী (যেমন: AGP-XXXX-XXXX-XXXX) তৈরি করবে
       ↓
৪. কাস্টমারের ইমেইলে লাইসেন্স কী এবং ডাউনলোড লিংক পাঠিয়ে দেওয়া হবে
       ↓
৫. সফটওয়্যার ওপেন করে লাইসেন্স কী দিলে পিসির Hardware ID সহ অ্যাক্টিভেট হয়ে যাবে
```

## ৩. অর্গানিক হোয়াইট হ্যাট এসইও (Organic SEO Strategy)

পিডিএফ এডিটর সফটওয়্যারটি যেন গুগল সার্চে সহজেই র‍্যাংক করতে পারে, তার জন্য আমরা নিম্নলিখিত হোয়াইট হ্যাট এসইও রুলস প্রয়োগ করব:

### ক) সফটওয়্যার স্কিমা মার্কআপ (SoftwareApplication Schema)
গুগল সার্চ বটকে এটি যে একটি ডেস্কটপ সফটওয়্যার তা বোঝাতে বিশেষ স্কিমা ডাটা যুক্ত করা হবে:
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  "name": "Antigravity PDF Pro",
  "operatingSystem": "Windows 10, Windows 11",
  "applicationCategory": "BusinessApplication",
  "offers": {
    "@type": "Offer",
    "price": "29.99",
    "priceCurrency": "USD"
  }
}
</script>
```

### খ) প্রোডাক্ট কি-ওয়ার্ড অপ্টিমাইজেশন (SEO Keywords)
সরাসরি সার্চ ভলিউম আছে এমন কি-ওয়ার্ড টাইটেল ও মেটা ডেসক্রিপশনে ব্যবহার করা হবে:
- *"desktop pdf editor windows 11", "offline pdf measurement tool", "auto redact pdf files locally", "bates numbering pdf editor"*।

### গ) হাই-স্পিড ল্যান্ডিং পেজ
ল্যান্ডিং পেজটিতে কোনো ভারী বা স্লো প্লাগইন থাকবে না। পিওর এইচটিএমএল, সিএসএস এবং মিনিফাইড জাভাস্ক্রিপ্ট ব্যবহার করা হবে যেন পেজ লোডিং স্পিড স্কোর গুগলে ৯০-এর বেশি থাকে।
