# ❓ Antigravity PDF Pro - FAQ & Troubleshooting

## ⚠️ সাধারণ সমস্যা এবং সমাধান

### **সমস্যা #1: "ffmpeg.dll was not found"**

#### কারণ:
- শুধুমাত্র .exe ফাইল কপি করেছেন
- DLL ফাইলগুলি আলাদা জায়গায় রেখেছেন

#### সমাধান:
```
✅ সঠিক উপায়:
1. সম্পূর্ণ "win-unpacked" ফোল্ডার খুঁজুন
2. পুরো ফোল্ডার একসাথে কপি করুন
3. Desktop বা অন্য জায়গায় পেস্ট করুন
4. ফোল্ডারের ভেতর থেকে .exe চালান

❌ ভুল উপায়:
- শুধু .exe কপি করবেন না
- .exe আলাদা জায়গায় রাখবেন না
```

---

### **সমস্যা #2: "The procedure entry point ... was not found"**

#### কারণ:
- Visual C++ Runtime নেই
- Windows আপডেট নেই

#### সমাধান:
```
1. Visual C++ Redistributable ডাউনলোড করুন:
   https://aka.ms/vs/17/release/vc_redist.x64.exe

2. ইনস্টল করুন

3. আপনার Windows আপডেট করুন:
   Windows Update > Check for updates

4. কম্পিউটার রিস্টার্ট করুন

5. অ্যাপ পুনরায় চেষ্টা করুন
```

---

### **সমস্যা #3: অ্যাপ খুব ধীর চলছে**

#### কারণ:
- বড় PDF ফাইল
- কম RAM
- অনেক সংখ্যক ট্যাব খোলা

#### সমাধান:
```
1. অ্যাপ রিস্টার্ট করুন
2. অপ্রয়োজনীয় ট্যাব বন্ধ করুন
3. বড় ফাইল ছোট করে চেষ্টা করুন
4. অন্য প্রোগ্রাম বন্ধ করুন

দীর্ঘমেয়াদী:
- RAM আপগ্রেড করুন (নূন্যতম 8GB)
- SSD ড্রাইভে রাখুন
```

---

### **সমস্যা #4: ফাইল সেভ হচ্ছে না**

#### কারণ:
- অপর্যাপ্ত স্টোরেজ
- ফাইল পারমিশন সমস্যা
- ভাইরাস প্রোটেকশন ব্লক করছে

#### সমাধান:
```
1. ডিস্ক স্পেস চেক করুন (কমপক্ষে 500MB খালি)
2. অন্য ফোল্ডারে সেভ করার চেষ্টা করুন
3. ভাইরাস প্রোটেকশন চেক করুন
   - Temporary disable করে চেষ্টা করুন
4. ফোল্ডার পারমিশন চেক করুন
   - Right-click > Properties > Security
```

---

### **সমস্যা #5: অ্যাপ কখনো স্টার্ট হয় না**

#### কারণ:
- কোর DLL ফাইল মিসিং
- System corruption
- Antivirus ব্লক করছে

#### সমাধান:
```
Step 1: চেক করুন প্রয়োজনীয় ফাইল আছে কিনা
- Antigravity PDF Pro.exe
- ffmpeg.dll
- d3dcompiler_47.dll
- libEGL.dll
- libGLESv2.dll

Step 2: প্যাকেজ পুনরায় ডাউনলোড করুন

Step 3: Antivirus ব্যতিক্রম যোগ করুন:
- Settings > Virus & threat protection
- Add exclusion > Antigravity folder

Step 4: Administrator হিসাবে চালান:
- Right-click on .exe
- Run as Administrator
```
